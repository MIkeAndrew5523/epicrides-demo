# Frontend Assets

## Color Palette
- Primary blue: `#003c6d`
- Accent orange: `#ff6900`
- Cyan blue: `#0693e3`
- Red: `#cf2e2e`
- Gray: `#abb8c3`
- White: `#ffffff`
- Dark text: `#333333`
- Light bg: `#f5f5f5`

## Font Stack
```css
font-family: system-ui, -apple-system, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;
```

## Breakpoints
- Mobile: max-width 768px
- Tablet: max-width 1024px
- Desktop: 1025px+

## Z-Index Layers
- Dropdown menu: 100
- Mobile nav overlay: 200
- Modal/overlay: 300

## Class Naming
Use descriptive hyphenated classes: `.hero-section`, `.nav-dropdown`, `.testimonial-slider`, `.booking-form`

## JS Patterns
- No framework — vanilla JS with DOM queries
- Use `fetch()` for API calls
- Event delegation where possible
